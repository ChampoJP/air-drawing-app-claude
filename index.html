📋 Instrucciones de Uso:

2. Mueve tu mano frente a la cámara para dibujar

3. Los movimientos rápidos crean trazos

4. Usa los controles para personalizar tu experiencia

5. El indicador rojo se activa cuando estás dibujando

🚀 Cómo subir a GitHub Pages (HTTPS garantizado):

  1. Crear repositorio:
    • Ve a github.com y crea una cuenta si no tienes
    • Haz clic en "New repository"
    • Nombra tu repositorio: air-drawing-app
    • Marca la opción "Public" y "Add README"
  2. Subir el archivo:
    • Copia todo este código HTML
    • En tu repositorio, haz clic en "Add file" > "Create new file"
    • Nombra el archivo: index.html
    • Pega todo el código
    • Haz clic en "Commit new file"
  3. Activar GitHub Pages:
    • Ve a "Settings" de tu repositorio
    • Busca la sección "Pages" en el menú izquierdo
    • En "Source", selecciona "Deploy from a branch"
    • Selecciona "main" branch y "/ (root)"
    • Haz clic en "Save"
  4. Acceder a tu app:
    • En unos minutos, tu app estará disponible en:
    • https://tu-usuario.github.io/air-drawing-app
    • ¡Con HTTPS automático para que funcione la cámara! 🎉

💡 Consejo: Una vez en GitHub Pages con HTTPS, la aplicación detectará automáticamente tu cámara y funcionará perfectamente.
